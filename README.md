# CafeMenu

**CafeMenuApp** is a mobile application that allows users to browse and filter a café menu, add items to their cart, and explore products in a clean, intuitive interface. The design is inspired by Apple’s **Liquid Glass** style, particularly for buttons and UI elements.

---

## Features

- **Filter Menu Items:** Quickly find drinks,
pastries, or other categories.  
- **Add to Cart:** Users can select items and manage their order.  
- **Clean, Modern Design:** Inspired by Apple’s Liquid Glass style for a polished look.

---

## Technologies Used

- **Frontend:** React Native, JavaScript  
- **Styling:** Custom styles for mobile UI, different from standard React web apps  
- **Platform:** Expo  

---

## Motivation & Learning

This was my first project in **React Native**, giving me hands-on experience with mobile app development and styling. Unlike a standard React web app, React Native required learning how to structure and style components specifically for mobile platforms.  

The project was an exciting opportunity to explore mobile UX, responsive design, and interactive UI elements. It was also a fun project to combine my growing coding skills with design inspiration from Apple’s Liquid Glass style.

---

## Achievements

- Learned React Native fundamentals and mobile-specific styling.  
- Built a fully functional, interactive menu app from scratch.  
- Explored modern design inspirations for a sleek user interface.
